# CityGML lod0FootPrint → GeoPackage (WGS84)

Bu Streamlit uygulaması, CityGML (.gml) dosyalarındaki lod0FootPrint geometrilerini
WGS84 (EPSG:4326)'e dönüştürerek GeoPackage (.gpkg) olarak dışa aktarır.

## Kurulum

pip install -r requirements.txt

## Çalıştırma

streamlit run app.py

## Deploy (Streamlit Cloud)

- GitHub'a yükleyin
- Streamlit Cloud → New App
- Repo seçin
- Main file: app.py
